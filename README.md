# Configuring a Virtual Environment in your Local Machine

## Create venv file
python -m venv venv

## Activating Virtual Environment (venv)
source venv/bin/activate

## Install requirements.txt
pip install -r requirements.txt

## Check Installed Libraries
pip freeze >> rrrr.txt
